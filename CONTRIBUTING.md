# Contributing to Kit Skills

Thank you for considering contributing to Kit Skills! To get started, please follow these steps:

## Contributor Setup Steps
1. Fork the repository on GitHub.
2. Clone your forked repository to your local machine:
   ```bash
   git clone https://github.com/{your-username}/kit-skills.git
   ```
3. Navigate to the project directory:
   ```bash
   cd kit-skills
   ```
4. Install the project dependencies:
   ```bash
   npm install
   ```

## Testing and Linting
To ensure code quality, make sure to run the following commands:
- To run tests:
  ```bash
  npm test
  ```
- To lint the code:
  ```bash
  npm run lint
  ```

## Commit and Branch Conventions
- Use meaningful commit messages that explain the changes clearly.
- For branching, use the following convention:
  - `feature/your-feature-name` for new features
  - `bugfix/your-bugfix-name` for bug fixes

## Running CI Locally
For information on how to run continuous integration (CI) locally, refer to the [CI Documentation](link-to-ci-docs).
